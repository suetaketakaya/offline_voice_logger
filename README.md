# OfflineVoiceLogger v1.0.0

完全オフラインで動作する音声文字起こしアプリケーション

## ダウンロード

**最新版**: [v1.0.0 - GitHub Releases](https://github.com/suetaketakaya/offline_voice_logger/releases/tag/v1.0.0)

**公式サイト**: [https://offlinevoicelogger-6fc29.web.app](https://offlinevoicelogger-6fc29.web.app)

### インストール方法

1. [GitHub Releases](https://github.com/suetaketakaya/offline_voice_logger/releases/latest)から最新版をダウンロード
2. `OfflineVoiceLogger_Setup_v1.0.0.exe` を実行
3. インストールウィザードに従ってインストール
4. デスクトップまたはスタートメニューから起動

## 特徴

- **完全オフライン動作**: インターネット接続不要
- **高精度文字起こし**: faster-whisper (large-v3モデル) による高精度な日本語・英語認識
- **リアルタイム処理**: 音声をリアルタイムで文字起こし
- **時刻データ表示**: 文字起こし結果に実際の時刻（HH:MM:SS形式）を表示
- **マイク & システムオーディオ対応**: マイク入力とPC画面の音声の両方に対応
- **スクリーンキャプチャ対応**: 画面録画、ブラウザ、Teams等の音声を文字起こし
- **自動保存**: 文字起こし結果を自動でテキストファイルに保存

## システム要件

- **OS**: Windows 10/11 (64bit)
- **メモリ**: 8GB以上推奨 (large-v3モデル使用時)
- **ディスク空き容量**: 5GB以上 (モデルファイル含む)
- **CPU**: Intel Core i5以上推奨

## 使い方

1. アプリケーションを起動
2. 音声デバイスを選択
3. 録音開始ボタンをクリック
4. 文字起こし結果がリアルタイムで表示されます

詳細は AUDIO_SETUP.md を参照してください。
